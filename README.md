Instrucciones de uso del taller de automatización.



\*\* 1\\. Preparación del proyecto: \*\*



Asegúrate de tener instalado Java y Gradle (preferiblemente 17 y 8.6, respectivamente) en tu equipo, y de que la variable de entorno JAVA\\\_HOME esté configurada correctamente.



\*\* 2\\. Ejecutar las pruebas: \*\*



Abre una terminal en la carpeta raíz del proyecto y ejecuta el siguiente comando:



"gradle clean test"



\*\* 3\\. Ver resultados: \*\*



Al finalizar la ejecución, SerenityBDD generará un reporte completo de las pruebas. Puedes acceder a él en la siguiente ruta:



“build/reports/tests/test/index.html”



Abre este archivo en un navegador para revisar los resultados, que incluyen información detallada sobre las pruebas exitosas, fallidas y cualquier error detectado.
